 The skills and knowledge you learn will enable you to:

Source, clean and transform data into descriptive summaries that can be presented, understood, and lead to action.
Apply foundational techniques and principles of computing and utilize mathematical and statistical models for decision making
Demonstrate ability to understand and explain database application design, data models, management, storage techniques, query languages, and database integration
Select and use appropriate tools for analysis of data
Apply oral, visual and written communication skills to collaborate effectively with key stakeholders and influence decision makers
